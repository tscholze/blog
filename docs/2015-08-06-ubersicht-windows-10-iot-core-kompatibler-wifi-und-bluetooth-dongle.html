Übersicht Windows 10 IoT Core kompatibler Wifi- und Bluetooth-Dongle

Mit der aktuellen Version (Juli’15) von Microsofts Windows 10 IoT Core hielt nun endlich der Support für Wifi- und Bluetooth-Dongle Einzug. Die Liste mit den unterstützen USB-Geräten an einem Raspberry Pi ist im Moment noch sehr überschaubar.

Wifi-Dongle:

  • Offizieller Raspberry Pi Wifi-Dongle

Bluetooth-Dongle:

  • Orico Model A Bluetooth Dongle (nicht bei Amazon D. verfügbar)

Für das MWM-Board schaut es etwas besser aus. Mehr dazu auf den offiziellen Seiten zu Wifi und Bluetooth.
